# djlt: the DJ’s library engine

Welcome to the documentation for **djlt**, the high-performance library management system for Rekordbox and Plex.

If you’re new to `djlt`, begin with the **[Getting Started](guide/getting-started.md)** guide. That guide walks you through installing `djlt`, setting up your Rekordbox XML path, and starting to query your library.

Then you can get a more detailed look at the query engine in the **[Syntax](query/syntax.md)** and **[Providers](providers/index.md)** references. You can also explore the full **[Command-Line Interface](commands/index.md)** reference to see everything `djlt` can do.

If you need help or want to report a bug, please file an issue in the **[issue tracker](https://github.com/llttlltt/dj-library-tools/issues)**.
