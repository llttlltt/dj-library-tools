# Workflows

Step-by-step guides for combining `djlt` commands to solve common library management problems.

- **[Automating Playlists](automating-playlists.md)**: Use track metadata to automatically manage "to-do" lists.
- **[Cleaning Up](cleaning-up.md)**: Find and prune unplayed or low-quality music.
- **[Library Repair](library-repair.md)**: Repair broken file paths and enrich playlist metadata.
- **[Metadata Hardening](metadata-hardening.md)**: Systematically identify and fix gaps in your track tags.
- **[Metadata Reconciliation](metadata-reconciliation.md)**: Synchronize grids, ratings, and tags between different libraries.
- **[Organizing by Complexity](organizing-by-complexity.md)**: Categorize tracks based on beatgrid complexity.
- **[Organizing Resources](organizing-resources.md)**: Manage the hierarchy of your library by moving playlists and folders.
- **[Syncing to Mobile](syncing-to-mobile.md)**: Export Plex or Rekordbox playlists to M3U8 files.
