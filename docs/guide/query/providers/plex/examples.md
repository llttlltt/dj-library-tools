# Plex Examples

**List a Plex playlist by name**
```bash
djlt list "plex/playlists:Summer"
```

**Sync a Plex playlist to Rekordbox**
```bash
djlt sync "plex/playlists:Summer" "rb/playlists:Plex Sync"
```

**Sync a Plex playlist to M3U8**
```bash
djlt sync "plex/playlists:Summer" "m3u8:/path/to/playlist.m3u8"
```
