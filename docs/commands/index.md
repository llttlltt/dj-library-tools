# djlt

DJ Library Tools CLI

### Options

```
  -h, --help         help for djlt
  -x, --xml string   Path to the Rekordbox XML library
```

A comprehensive CLI tool for managing DJ libraries and Rekordbox XMLs.
## See also

* [djlt auth](auth.md)	 - Authenticate with third-party providers
* [djlt config](config.md)	 - View or update application configuration
* [djlt folder](folder.md)	 - Manage rekordbox playlist folders
* [djlt list](list.md)	 - List items from a location (e.g. plex/playlists:Summer)
* [djlt metadata](metadata.md)	 - Manage track metadata between Rekordbox XML libraries
* [djlt playlist](playlist/index.md)	 - Manage rekordbox playlists
* [djlt stat](stat.md)	 - Show statistics for tracks matching the query
* [djlt sync](sync.md)	 - Sync items between a source and target
