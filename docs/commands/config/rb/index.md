# djlt config rb

Configure Pioneer Rekordbox settings.

## Subcommands

### file
Set or get the path to your primary Rekordbox XML export.
```bash
djlt config rb file [path]
```

## Options

```
  -h, --help   help for rb
```

## Inherited Options

```
      --apply         Actually apply changes to the library (destructive)
  -f, --file string   Path to the primary library file (Rekordbox XML, M3U, etc.)
  -v, --verbose       Enable verbose logging
```
