# folder

Manage rekordbox playlist folders

```
djlt folder [rb/folders query] [flags]
```

### Options

```
      --dry-run         Preview changes without writing
  -h, --help            help for folder
      --move string     Move matched folder into this parent folder
      --new string      Create a new folder with this name
      --parent string   Parent folder for --new (default: root level)
      --remove          Remove matched folder
      --rename string   Rename matched folder to this name
```

### Inherited Options

```
  -x, --xml string   Path to the Rekordbox XML library
```
