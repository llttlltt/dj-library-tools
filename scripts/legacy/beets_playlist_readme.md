# Local

### 1. Fix Paths

```bash
./fix_playlist_extensions.sh "./Playlists/PLAYLIST_NAME.m3u"

# ./fix_playlist_extensions.sh "./Playlists/DJ Intro Outro.m3u"
# ./fix_playlist_extensions.sh "./Playlists/DJ Crate #25.m3u"
# ./fix_playlist_extensions.sh "./Playlists/DJ Crate #26.m3u"
# ./fix_playlist_extensions.sh "./Playlists/DJ Yeet #26Q2.m3u"
```

### 2. Convert to M3U8

```bash
./m3u-to-m3u8.sh "./Playlists/PLAYLIST_NAME.m3u"

# ./m3u-to-m3u8.sh "./Playlists/DJ Intro Outro.m3u"
# ./m3u-to-m3u8.sh "./Playlists/DJ Crate #25.m3u"
# ./m3u-to-m3u8.sh "./Playlists/DJ Crate #26.m3u"
# ./m3u-to-m3u8.sh "./Playlists/DJ Yeet #26Q2.m3u"
```
